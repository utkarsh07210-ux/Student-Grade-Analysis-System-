# Student Grade Analysis System

A Python data analysis tool that processes student marks, assigns grades, and visualizes performance using Pandas and Matplotlib.

## Features
- Reads student marks from Excel
- Calculates total, average, and letter grade (A+ to F) per student
- Generates 3 charts:
  - Bar chart — average marks per student
  - Pie chart — grade distribution
  - Bar chart — subject-wise class average
- Prints a summary report (class average, topper, lowest scorer)
- Saves charts as PNG files

## Tech Stack
'Python' 'Pandas' 'Matplotlib'
